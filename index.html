Components of a Search Engine

Robot

Robot is a program that automatically traversees the hypertext structure of the web by retriving a web page document and following the hyperlinks on the page. In addition, some robots may have access and store the title, meta log scription and some text on the page.

Database

The collection of information that is organized so that its content can be accessed, managed and updated. The web page that displays the results of your search has information from the database that is used by the search engine site. Some search engines use some of its content recieved from other search engines. SUch as Yahoo gets its search engine results from Google.

Search Form

The graphical user information that allows a user to type in a word or phrase to search for. This is where the user types in keywords related to their search. When submitted to the keywprdds are sent to a script that searches the database using the keywords.The rsults set is formatted with a link to each page, along with additional information that may include the page title, a brief description the first few lines of text, or the size of the page. This varies by search engine.

SearchEngine Optimation Tips

Keywords

Spend time brainstorming terms and pharses that people may use when searching for your site. These terms are what describe your website or business.

Page Titles

A description page title which include your company name or website name will help your site market itself. The page title is often displayed in the SERP and saved when a visitor bookmarks your page. DO not use the exact same title for each page of your website. Include keywords in the next page title that are appropraite for the page.

Heading Tags

Use structures tags, h1, h2, etc, to organize your content. If approproate include some keywords in the text contained with the heading tags. Some search engines will give higher list position if keywrds are used in the page title or heading. However, do not spam keywords, meaning repeating the words over and over within the content of the page. Search engines continue to become more sophisticated and spaming can prevent your page from being listed if it percieves you are trying to cheat the system.

Description

Write a few sentences about your website or business that makes it special and why people would want to visit. The description should be inviting and interesting so that someone would choose your site.

Decription Meta Tag

Self-contained tage that is placed in the head section of a web page and is displayed on the SERP by some search engines, such as Google.

Linking

Verify that all hyperlinks are working.Each page on your website should be reachable by a text hyperlink and should be descriptive. DO not use phrases such as more info or click here. SHould use keywords as appropriate.

Images and Multimedia

Search engine robots do not see the text within your images. COnfigure meaningful alt text for images and include relevant keywords

Valid Code

While it is not required by search engines that your code be validiated, well structured and valid code is more easily processed by search engines robots which could help your placement in the search engine.

Content of Value

Your website should contain high-quality, well-organized content that is of value to your visitors.

Additional Techniques for SEO

Map your Site

An HTML site map is a web page with a map of your site that containes hierachical list of hyperlinks to the major pages in your website. This can be helpful for website visitors and assist search engine robots as they follow hyperlinks on your site. An XML site map is an XML file that us used to earch engines only, not visitors. An XML site map provides tha following transformation about each page to a search engine

  • date that each page was last modified
  • indicator of how frequently each page changes
  • priority level for each page
Link Popularity

Search engines rate websites based on the number of sites that link to a particular website and the quality of those sites. Link popularity can determine its order in the SERP.

SMO - Social Media Optimization

Make it easy for your current and potential website visitors by creating content of value that is sharable. SUch as providing links to Twitter, Facebook and Pinterest.
